# 🧠 MindReliever – Student Mental Wellness Management System

## 📖 Overview
MindReliever is a software system designed to support student mental well-being by providing self-assessment tools, mood tracking, personalized recommendations, and wellness resources.

## 🎯 Objectives
- ✅ Enable students to assess their mental wellness.
- 📊 Allow daily mood tracking and analytics.
- 💡 Provide personalized wellness recommendations.
- 📚 Maintain a library of mental health resources.
- 👨‍💼 Support administrators and counselors.
- 📈 Generate anonymized wellness trends.
